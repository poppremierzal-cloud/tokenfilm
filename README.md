# ТОКЕН: ФИЛЬМ — лендинг фестиваля

Готовый интерактивный лендинг для размещения на домене.

## Состав файлов

```
index.html                      ← главная страница (всё внутри: HTML + CSS + JS)
token-logo.svg                  ← логотип на фоне
token-logo-transparent.svg      ← логотип без фона (для футера)
```

Все три файла кладутся в корень сайта рядом друг с другом.

## Что НУЖНО настроить разработчикам

### 1. Отправка формы заявки в Telegram (обязательно)

В файле `index.html`, в самом конце, найдите строку:

```js
const TELEGRAM_ENDPOINT = '/api/apply';  // ← ЗАМЕНИТЕ НА ВАШ URL
```

Замените на URL вашего бэкенд-эндпоинта, который пересылает данные в Telegram.
Подробная инструкция и пример кода на Node.js — прямо там же в комментарии.

**Почему нельзя зашить токен бота прямо в HTML:** HTML-код виден всем
посетителям сайта, токен бота утечёт за минуты, и любой сможет
рассылать сообщения от имени бота. Поэтому нужен серверный прокси
(минимум 10 строк кода на любом языке).

Альтернативы без своего бэкенда:
- **Formspree** (formspree.io) — вписать URL вида `https://formspree.io/f/xxxxxx`
- **Getform** (getform.io) — аналогично
- **FormSubmit** (formsubmit.co) — без регистрации

Заявки сначала придут на email, оттуда — встроенными интеграциями
в Telegram (через IFTTT, Zapier или просто пересылкой).

### 2. Положение о фестивале (PDF)

В файле `index.html` найдите блок с `id="modal"`. Внутри есть
`<div class="modal__placeholder">` — замените его на:

```html
<iframe src="polozhenie.pdf" style="width:100%;height:600px;border:1px solid #000;"></iframe>
```

И положите файл `polozhenie.pdf` в корень сайта.

### 3. Соцсети — реальные ссылки

В футере (внизу страницы) найдите блок «Соцсети» и замените `href="#"`
на реальные ссылки на Telegram, ВКонтакте, YouTube.

### 4. Список городов

В секции «Площадки фестиваля» (id="venues") список городов в
комментариях помечен как редактируемый. Просто измените названия
в блоках `<div class="city">...</div>`.

### 5. Жюри

В секции «Жюри фестиваля» — 5 карточек с плейсхолдерами «Имя · скоро».
Когда будут утверждены члены жюри (до 25 мая 2026), замените:
- блок `<div class="jury-card__ph"></div>` на `<img src="jury/photo.jpg" />`
- текст в `<span class="name">` на реальное имя

### 6. Партнёры

В секции «Партнёры» — 8 пустых слотов с надписью «Партнёр 01» и т.д.
Замените содержимое каждого `<div class="partner-slot">` на:

```html
<a href="https://partner-site.ru" class="partner-slot">
  <img src="partners/logo.svg" alt="Название" style="max-height:60%;max-width:70%;" />
</a>
```

### 7. OpenGraph (для красивых ссылок в соцсетях)

В `<head>` найдите теги `<meta property="og:..."` и замените URL картинки
на абсолютный путь к большому превью (1200×630px).

## Технические детали

- Чистый HTML/CSS/JS, никаких сборок и фреймворков не нужно
- Шрифты подгружаются с Google Fonts (Unbounded, Manrope, Cormorant Garamond, IBM Plex Mono)
- Адаптивен от 320px и шире
- SEO-теги, OpenGraph, мета-описания на месте
- Анимации только на CSS + IntersectionObserver (без библиотек)

## Поддержка браузеров

Современные браузеры (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+).
В IE не работает (и не должен).
